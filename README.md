# ⚡ TypeLikeFlash

A fast-paced typing speed test application built with React. Test your typing skills and track your performance with real-time feedback.

![Typing Speed Test](https://user-images.githubusercontent.com/71912675/205044101-a72afd80-ddcc-44c5-960a-5cf1b22f58fe.png)

## 🚀 Live Demo

**[Try it now!](https://sapscode.github.io/type-like-flash/)**

## ✨ Features

- ⏱️ Real-time WPM (Words Per Minute) calculation
- 📊 Track characters typed, words, and speed
- 🎯 Instant visual feedback (green for correct, red for incorrect)
- 60-second timed challenges
- 📱 Responsive design
- 📢 Share results on social media

## 🛠️ Tech Stack

- React 17 with Hooks
- CSS3 animations
- Typewriter Effect library
- GitHub Pages deployment

## 🔧 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## 📁 Project Structure

```
src/
├── components/        # React components
├── Assets/           # Images and icons
├── data/             # Sample paragraphs
└── index.js          # Entry point
```

## 📝 How It Works

1. Start typing to begin the 60-second timer
2. Get instant feedback on each keystroke
3. View your final stats (WPM, words, characters) when time's up
4. Share your score or try again
